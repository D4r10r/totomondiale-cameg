# Totomondiale Cameg

Sito statico gratuito per GitHub Pages.

- `index.html`: classifica Posizione / Nome / Punti.
- `pronostici.html`: riepilogo pronostici compatto stile Excel.
- `data/pronostici.json`: pronostici dal file Excel.
- `data/risultati-auto.json`: risultati aggiornati da GitHub Actions.
